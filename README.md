

# Studymonk Assignment
site: https://github.com/GauravRajpt/studymonk-assignment


This project is a recruitment solution website developed using HTML, CSS, and JavaScript. It provides a platform for users to register, login, and search for candidates based on location and job role.

## Features

- **User Registration**: Users can create an account by filling out the registration form. The registration form collects necessary information such as name, email, and password.

- **User Login**: Registered users can log in using their email and password. Upon successful login, they are redirected to the search candidate page.

- **Candidate Search**: The search candidate page allows users to search for candidates based on their location and job role. Users can enter specific criteria to filter the candidate list.

- **Candidate List**: The search page displays a list of candidates based on the search criteria. Each candidate is presented in a box format.

- **Hover Effect**: When hovering over a candidate's box, it moves slightly, providing a visual feedback to the user.

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

1. Register a new user account by filling out the registration form.

2. Log in using the registered email and password.

3. You will be redirected to the search candidate page.

4. Enter the desired location and job role in the search form to filter the candidate list.

5. Hover over a candidate's box to see the hover effect.
